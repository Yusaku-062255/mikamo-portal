# みかもポータル アクセスガイド

## 📱 このポータルについて

みかもポータルは、株式会社ミカモの4事業（SOUP / M-NET / ミカモ石油 / ミカモ喫茶）と本部をつなぐDXツールです。

### 各事業にとっての価値

- **SOUP（カーコーティング）**: AIが洗車からコーティングへのアップセル方法を提案。売上トレンドで受注率の改善を可視化
- **M-NET（中古車販売）**: AIが来店から成約までのボトルネックを発見。在庫回転率を意識した提案
- **ミカモ石油（ガソリンスタンド）**: AIが給油客の送客動線を提案。天気と売上の相関を可視化
- **ミカモ喫茶（飲食・カフェ）**: AIが客単価アップの方法を提案。滞在満足度と回転率のバランスを意識

## 🌐 アクセス方法

### 1. 初回アクセス

1. 管理者から共有されたURLを開きます
   - 例: `https://mikamo-portal-frontend-xxxxx-an.a.run.app`
   - 最初のURLは長いですが、**Google公式の安全なURL**です

2. ログイン画面が表示されます

### 2. スマホに「アプリ」として追加する（推奨）

#### iPhone（Safari）

1. SafariでURLを開く
2. 画面下部の「共有」ボタン（□↑）をタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ
5. ホーム画面にアイコンが追加されます

#### Android（Chrome）

1. ChromeでURLを開く
2. メニュー（⋮）をタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ
5. ホーム画面にアイコンが追加されます

これで、ブラウザを開かなくても、アプリアイコンから直接アクセスできます！

## 📝 主な機能

### ダッシュボード

- 時間帯に応じた挨拶
- 今週の頑張り（売上、投稿回数、お客様数、取引数）
- クイックアクション（振り返り入力、AI相談）

### 今日の振り返り

- 天気選択（晴れ・曇り・雨・雪）
- 数値入力（売上、お客様数、取引数）
- 良かったこと・課題・メモの記録
- 保存時に「お疲れ様でした！」のアニメーション

### AI相談

- 部署に応じた質問サジェスト
- 直近のデータを踏まえた具体的なアドバイス
- 音声入力にも対応

### マネージャー向け機能

- 売上・客数トレンドグラフ（14日間）
- 部署間比較グラフ（本部・経営向け）
- スタッフへのコメント機能

## 💡 よくある質問

### Q: URLが長くて覚えられない

### Q: パスワードを忘れた

A: 管理者に連絡してください。

### Q: エラーが表示される

A: ページを再読み込みしてみてください。それでも解決しない場合は、管理者に連絡してください。
